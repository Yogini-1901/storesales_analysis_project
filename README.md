# Store Sales Analysis Project

## 📊 Project Overview

This project analyzes the sales performance of an online store.
The goal is to understand sales trends, identify profitable and loss-making products, and support business decisions using data.

## 🎯 Objective

* Analyze sales data of an online store.
* Identify top-performing categories and regions.
* Detect loss-making sub-categories.
* Provide insights to improve overall sales performance.

## 🛠 Tools & Technologies Used

* Python (Pandas, NumPy)
* SQL
* Power BI
* Excel

## 📂 Dataset

The dataset contains information about:

* Order ID
* Customer Name
* Region
* Category
* Sub-Category
* Sales
* Profit

## 📈 Analysis Performed

* Data Cleaning and Preparation
* Sales Aggregation using SQL
* Profit and Loss Analysis
* Regional Sales Comparison
* Dashboard creation in Power BI

## 📊 Power BI Dashboard Insights

* Technology category generated the highest profit.
* Some sub-categories showed losses.
* The West region recorded the highest sales.
* Sales trends help identify areas for business improvement.
